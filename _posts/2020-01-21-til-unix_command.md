 **Unix command**

기본적인 유닉스명령어


 
 1. 그 다음 경로로 가기 (디렉토리 이동)

    - cd 파일이름

    ```
    cd user
    ```



2. 그 전 경로로 가기(디렉토리 이동)
    - cd..

    ```
    cd..
    ```



3. 파일이나 디렉토리 보기

    ```
    ls -a       //전부
    ls -l       //상세정보
    ls -s       //크기별정렬
    ls -h       //단위 표현 변경
    ls -al      //상세, 모두 출력
    ```


4. 명령어도움말

    ```
    man ls         //ls에 대한 도움말
    ```



5. 현재 디렉토리 경로 확인

    ```
    pwd
    ```


6. 화면 클리어

    ```
    clear
    ```
